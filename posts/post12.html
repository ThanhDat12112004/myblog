Hướng Dẫn Loop Through Object trong Javascript ES6

25 Tháng 12, 2019 10 phút đọc

Bài này nằm trong loạt bài Kỹ thuật xử lý Javascript nâng cao (đang cập nhật) trên trang blog chính thức trungquandev.com

Mục Tiêu

Chúng ta sẽ cùng nhau tìm hiểu các cách khác nhau để lặp qua một đối tượng trong Javascript, từ đó lựa chọn phương pháp phù hợp cho từng trường hợp cụ thể.

Các Phương Pháp Loop Through Object

1. Sử dụng vòng lặp for...in

Vòng lặp for...in giúp lấy được key của object trong mỗi lần lặp, và có thể dùng key đó để lấy value của object.


const object1 = {
  id: 17,
  name: 'thanhdat',
  website: 'https://thanhdatdev.com'
}

for (let key in object1) {
  if (object1.hasOwnProperty(key)) {
    console.log(\`\${key}: \${object1[key]}\`)
  }
}
// expected output:
// 'id: 17'
// 'name: thanhdat'
// 'website: https://thanhdatdev.com'

2. Sử dụng Object.keys

Object.keys() sẽ nhận object của chúng ta làm tham số và trả về một mảng chứa tất cả các key của object.


const object1 = {
  id: 17,
  name: 'thanhdat',
  website: 'https://thanhdatdev.com'
}

console.log(Object.keys(object1))
// expected output: ["id", "name", "website"]

Object.keys(object1).forEach(key => {
  console.log(\`\${key}: \${object1[key]}\`)
})
// expected output:
// 'id: 17'
// 'name: thanhdat'
// 'website: https://thanhdatdev.com'

3. Sử dụng Object.values

Object.values() trả về một mảng chứa tất cả giá trị của object mà không cần quan tâm đến key.


const object1 = {
  id: 17,
  name: 'thanhdat',
  website: 'https://thanhdatdev.com'
}

console.log(Object.values(object1))
// expected output: [17, "thanhdat", "https://thanhdatdev.com"]

Object.values(object1).forEach(value => {
  console.log(value)
})
// expected output:
// 17
// 'thanhdat'
// 'https://thanhdatdev.com'

4. Sử dụng Object.entries

Object.entries() trả về một mảng chứa các cặp key, value của object dưới dạng mảng con.


const object1 = {
  id: 17,
  name: 'thanhdat',
  website: 'https://thanhdatdev.com'
}

console.log(Object.entries(object1))
// expected output: [["id", 17], ["name", "thanhdat"], ["website", "https://thanhdatdev.com"]]

Object.entries(object1).forEach(([key, value]) => {
  console.log(\`\${key}: \${value}\`)
})
// expected output:
// 'id: 17'
// 'name: thanhdat'
// 'website: https://thanhdatdev.com'

5. Sử dụng Object.getOwnPropertyNames

Object.getOwnPropertyNames() trả về một mảng chứa tất cả tên thuộc tính của object.


const object1 = {
  id: 17,
  name: 'thanhdat',
  website: 'https://thanhdatdev.com'
}

console.log(Object.getOwnPropertyNames(object1))
// expected output: ["id", "name", "website"]

Object.getOwnPropertyNames(object1).forEach(prop => {
  console.log(\`\${prop}: \${object1[prop]}\`)
})
// expected output:
// 'id: 17'
// 'name: thanhdat'
// 'website: https://thanhdatdev.com'
