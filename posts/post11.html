Tìm hiểu về Callback Function trong JavaScript

11 Tháng 12, 2020 10 phút đọc

Giới thiệu

Callback function là một kỹ thuật được sử dụng trong JavaScript và lập trình chức năng. Bạn có thể sử dụng callback function để thực thi một đoạn mã sau khi một hành động khác đã hoàn thành.

Callback function là gì?

Callback function là một hàm được truyền vào như một tham số trong một hàm khác và được gọi khi hàm đó thực thi xong.

Ví dụ cơ bản về Callback function:


function notify(){
  console.log("Hello world");
}

function taskOne(callback){
  // Gọi notify function
  callback();
}

// Truyền function notify dưới dạng biến vào taskOne
taskOne(notify);

Callback function trong jQuery:


$("#btn_1").click(function() {
  alert("Btn 1 Clicked");
});

Callback function hoạt động như thế nào?

Ta có thể truyền một function như là một biến và return nó trong một function khác. Khi callback function được truyền dưới dạng tham số, nó chỉ được thực thi khi hàm chứa nó gọi tới.

Callback function là Closure

Closure cho phép một function truy cập vào scope của function cha, và điều này cũng đúng đối với callback function. Nó có thể truy cập các biến trong phạm vi của function chứa nó.

Cách gọi Callback function

Sử dụng anonymous functions


$("#btn_1").click(function() {
  alert("Btn 1 Clicked");
});

Sử dụng một callback function đã được đặt tên


function namedCallback(){
   alert("namedCallback()");
}
function testFunction(callback){
   callback();
}
testFunction(namedCallback);

Truyền tham số tới callback function


var author = "Thanh Dat";
function whoWriteThis(param){
   alert("whoWriteThis() called by " + param);
}

function testFunction(callback){
   callback(author);
}

testFunction(whoWriteThis);

Gọi nhiều callback function


var someUrl = "https://example.com";

function successCallback(){
   // success code
}

function completeCallback(){
   // complete code
}

function errorCallback(){
   // error code
}

$.ajax({
   url: someUrl,
   success: successCallback,
   complete: completeCallback,
   error: errorCallback
});

Kết luận

Callback function là một công cụ mạnh mẽ trong JavaScript, giúp bạn xử lý các tác vụ bất đồng bộ và cải thiện hiệu quả code của mình. Việc hiểu rõ cách hoạt động của callback function sẽ giúp bạn làm việc với JavaScript một cách hiệu quả hơn.
